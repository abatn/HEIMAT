# HEIMAT – Fortschritt

## STRATEGIE (aus MD-Dateien)

### Projekt-Ziel (aus project-prompt.md)
"Entwicklung einer digitalen Plattform (Super App), die als zentrale Anlaufstelle für den deutschen Alltag dient."

### Kernprinzipien (aus README.md + project-prompt.md)
1. 100% Open Source
2. 100% Kostenfrei
3. 100% Legal
4. 100% Datenschutzkonform (DSGVO)
5. 100% Community-getrieben

### Technologie (aus project-prompt.md + README.md)
- Frontend: Flutter
- Backend: Node.js + Express
- Datenbank: PostgreSQL
- Cloud: Hetzner Cloud (€5/Monat)

### Features die funktionieren müssen (aus README.md)
**Mobilität:**
- Interaktive Karte mit OpenStreetMap
- ÖPNV-Verbindungssuche (GTFS)
- Routing für Fuß, Rad und Auto

**Finanzen:**
- P2P-Zahlungen über GNU Taler
- Keine BaFin-Lizenz nötig

**Gesundheit:**
- Arzt-Suche nach Fachrichtung und Ort
- Terminbuchung mit verfügbarer Zeitplanung

### Test-Checkliste (aus docs/endbenutzer-testing.md)
- [ ] Karte lädt korrekt
- [ ] ÖPNV-Suche funktioniert
- [ ] Guthaben wird angezeigt
- [ ] Zahlung funktioniert
- [ ] Ärzte werden angezeigt
- [ ] Terminbuchung funktioniert
