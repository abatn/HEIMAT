# HEIMAT 2.0 – Endbenutzer-Testing

## Für Web-App (Browser)

### Schritt 1: App öffnen
1. Öffne den Browser (Chrome, Firefox, Safari)
2. Gehe zu: https://abatn.github.io/HEIMAT/
3. Die App lädt automatisch

### Schritt 2: Features testen

#### Mobilität
1. Klicke auf "Mobilität" unten
2. Gib Start und Ziel ein
3. Klicke auf Suchen
4. Prüfe die Verbindungen

#### Finanzen
1. Klicke auf "Finanzen" unten
2. Prüfe das Guthaben
3. Klicke auf "Geld senden"
4. Gib Empfänger und Betrag ein

#### Gesundheit
1. Klicke auf "Gesundheit" unten
2. Suche nach einem Arzt
3. Klicke auf "Termin buchen"
4. Fülle das Formular aus

### Schritt 3: Feedback geben
- Erstelle ein GitHub Issue: https://github.com/abatn/HEIMAT/issues
- Oder schreibe im Matrix-Room: https://matrix.to/#/heimat:matrix.org

## Häufige Probleme

### App lädt nicht
- Prüfe die Internetverbindung
- Versuche einen anderen Browser
- Leere den Browser-Cache

### Karte zeigt nichts an
- Aktiviere den Standort
- Prüfe die Internetverbindung
- Versuche einen anderen Browser

### Zahlung funktioniert nicht
- Prüfe das Guthaben
- Prüfe den Empfänger-Username
- Versuche einen kleineren Betrag
