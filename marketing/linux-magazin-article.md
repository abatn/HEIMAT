## Artikelvorschlag: HEIMAT 2.0

### Titel
HEIMAT 2.0 – Eine Open-Source Super App für den deutschen Alltag

### Untertitel
Datenschutzkonform, kostenlos und auf Open-Source-Technologien basierend

### Abstract (max. 200 Zeichen)
HEIMAT 2.0 vereint Mobilität, Finanzen und Gesundheit in einer Open-Source App. On-Device AI und DSGVO-konformes Design machen die Plattform einzigartig.

### Zielgruppe
Linux-Nutzer, Entwickler, Datenschutzbewusste, Open-Source-Community

## Artikelstruktur

### 1. Einleitung (250 Wörter)
- **Problem:** Viele Apps für verschiedene Alltagsdienste
- **Lösung:** HEIMAT 2.0 als zentrale Plattform
- **Differenzierung:** 100% Open Source, DSGVO-konform, kostenlos
- **Zielgruppe:** Alle, die Wert auf Datenschutz und Transparenz legen

### 2. Features (500 Wörter)

#### Mobilität
- OpenStreetMap für Karten
- GTFS für ÖPNV-Daten
- OpenRouteService für Routing
- Kein Ticketkauf – nur Fahrpläne

#### Finanzen
- GNU Taler für P2P-Zahlungen
- Keine BaFin-Lizenz nötig
- Lightning Network als Backup
- Privacy by Design

#### Gesundheit
- Cal.com für Terminbuchung
- Keine TI-Anbindung
- Keine Patientendaten
- Manuelle Bestätigung durch Praxis

#### AI
- On-Device Sprachsteuerung (Vosk)
- Verspätungsvorhersage (LightGBM)
- Code-Generierung (Code Llama)

### 3. Technologie (400 Wörter)

#### Frontend
- Flutter (Cross-Platform)
- Dart
- Material Design

#### Backend
- Node.js + Express
- TypeScript
- REST API

#### Datenbank
- PostgreSQL
- Redis (Caching)

#### Infrastruktur
- Hetzner Cloud (deutscher Anbieter)
- Docker + Kubernetes
- GitHub Actions (CI/CD)

#### Kommunikation
- Matrix (E2E-verschlüsselt)
- Element als Client

### 4. Datenschutz (300 Wörter)

#### On-Device AI
- TensorFlow Lite
- Keine Datenübertragung
- Lokale Verarbeitung

#### DSGVO-Compliance
- Privacy by Design
- Datenminimierung
- Opt-in für AI-Funktionen
- Recht auf Löschung

#### Transparenz
- Open-Source-Code
- Community-Reviews
- Keine versteckten Algorithmen

### 5. Community (200 Wörter)
- GitHub-Repository
- Good-First-Issues
- Matrix-Room
- CONTRIBUTING.md

### 6. Fazit (150 Wörter)
- Ausblick auf Mini-App-Ökosystem
- Call-to-Action: Mitmachen
- Links zu Repository und Community

### Gesamtumfang: ca. 1.800 Wörter
