# HEIMAT 2.0 – Mastodon Account Einrichtung

## Schritt-für-Schritt-Anleitung

### 1. Server auswählen

**Optionen:**

| Server | Vorteile | Nachteile |
|--------|----------|-----------|
| **social.heimat-app.de** | Eigener Server, volle Kontrolle | Hosting-Kosten (~€5/Monat) |
| **mastodon.social** | Größter Server, viele Nutzer | Kein eigener Server |
| **fosstodon.org** | Open-Source-fokussiert | Weniger Nutzer |

**Empfehlung:** Zuerst auf mastodon.social starten, später migrieren.

### 2. Account erstellen

1. **Website öffnen:** https://mastodon.social (oder gewählter Server)
2. **"Sign Up"** klicken
3. **E-Mail-Adresse** eingeben
4. **Benutzername** wählen: `@heimat`
5. **Passwort** setzen
6. **E-Mail** bestätigen

### 3. Profil vervollständigen

#### Avatar
- **Bild:** HEIMAT-Logo hochladen
- **Größe:** 400x400 Pixel

#### Header
- **Bild:** Screenshot der App oder Cityscape
- **Größe:** 1500x500 Pixel

#### Bio (max. 500 Zeichen)

✅ 100% DSGVO-konform

Mobilität | Finanzen | Gesundheit | AI

GitHub: github.com/abatn/HEIMAT
Matrix: matrix.to/#/heimat:matrix.org
